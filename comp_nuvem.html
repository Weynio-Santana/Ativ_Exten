Computação em Nuvem

Descubra como a infraestrutura que alimenta a tecnologia moderna funciona.

O que é Computação em Nuvem?

Computação em Nuvem (Cloud Computing) é a entrega de serviços de computação — como servidores, armazenamento, bancos de dados, redes, software e inteligência — pela Internet ("a nuvem"). Em vez de possuir e manter sua própria infraestrutura de TI, você pode acessar esses serviços de um provedor de nuvem, como a AWS, Azure ou GCP. Isso oferece inovação mais rápida, recursos flexíveis e economias de escala. Os principais modelos de serviço são IaaS (Infraestrutura como Serviço), PaaS (Plataforma como Serviço) e SaaS (Software como Serviço).

AWS (Amazon Web Services)

Sobre AWS: Lançada pela Amazon em 2006, a AWS é a plataforma de nuvem mais adotada e abrangente do mundo. Oferece mais de 200 serviços completos de data centers globais. Milhões de clientes — incluindo as startups que mais crescem, as maiores corporações e os principais órgãos governamentais — usam a AWS para reduzir custos, ficar mais ágeis e inovar mais rapidamente. Seus serviços mais conhecidos incluem o EC2 (servidores virtuais), S3 (armazenamento de objetos) e Lambda (computação sem servidor).

Ideias de Projetos com AWS:

  • Hospedar um site estático usando o serviço S3.
  • Criar uma aplicação web simples e implantá-la em uma instância EC2.
  • Desenvolver uma API sem servidor (serverless) usando API Gateway e Lambda.
  • Configurar um banco de dados relacional com o serviço RDS.

Microsoft Azure

Sobre Azure: O Microsoft Azure é o serviço de computação em nuvem da Microsoft. Ele oferece uma gama enorme de serviços, incluindo os para computação, análise, armazenamento e rede. O Azure é conhecido por sua forte integração com tecnologias Microsoft e sua grande presença no mercado corporativo. Seus serviços equivalentes aos da AWS incluem Virtual Machines, Blob Storage e Azure Functions.

Ideias de Projetos com Azure:

  • Publicar uma aplicação ASP.NET Core no Azure App Service.
  • Criar uma função lógica (Logic App) para automatizar um fluxo de trabalho.
  • Armazenar e gerenciar arquivos no Azure Blob Storage.
  • Treinar um modelo de machine learning usando o Azure Machine Learning.

GCP (Google Cloud Platform)

Sobre GCP: O Google Cloud Platform é a suíte de serviços de nuvem do Google, oferecida na mesma infraestrutura que o Google usa internamente para seus produtos finais, como o Buscador Google e o YouTube. O GCP é altamente reconhecido por suas forças em computação de ponta, análise de dados (Big Data), machine learning e contêineres (com Kubernetes, originalmente desenvolvido pelo Google). Seus serviços principais incluem Compute Engine, Cloud Storage e Cloud Functions.

Ideias de Projetos com GCP:

  • Analisar um grande conjunto de dados usando o BigQuery.
  • Implantar uma aplicação em contêineres no Google Kubernetes Engine (GKE).
  • Criar uma Cloud Function para processar uploads de arquivos no Cloud Storage.
  • Utilizar as APIs de Machine Learning do Google (ex: Vision AI, Natural Language AI).

Terraform (Infraestrutura como Código)

Sobre Terraform: Terraform é uma ferramenta de código aberto para "Infraestrutura como Código" (IaC), criada pela HashiCorp. Em vez de configurar a infraestrutura manualmente (clicando em painéis), você escreve código para descrever os componentes da sua nuvem (servidores, bancos de dados, redes). O Terraform então lê esse código e constrói a infraestrutura descrita em qualquer provedor de nuvem (AWS, Azure, GCP, etc.). Isso torna o gerenciamento da infraestrutura mais seguro, repetível e colaborativo.

Ideias de Projetos com Terraform:

  • Escrever um script para criar uma rede virtual (VPC) e uma sub-rede.
  • Automatizar a criação de um servidor web em qualquer nuvem.
  • Gerenciar a configuração de um bucket S3 ou Blob Storage via código.
  • Criar um módulo reutilizável do Terraform para uma arquitetura comum.
